The first question the user must answer is cut off time in seconds. Input has set to be 1 to 18 seconds.

Second question is to ask the use to choose, who wants to go first - 1 for User and 2 for AI.

Input is handled by row/column table -> A1 = board[0][0],etc.

When game is won by either party a "win" message will pop and game will be terminated.
